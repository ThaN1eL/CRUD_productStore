### Setup .env file

```js
PORT=3000
PGUSER = ...
PGPASSWORD = ...
PGHOST = ...
PGDATABASE = ...

ARCJET_KEY = ...
ARCJET_ENV = development

ARCJET (https://arcjet.com/)
PG data from Neon POSGRESQL
```

### Run the API

```shell
"npm run dev"
```

### Run the Frontend

```shell
"cd frontend" and then "npm run dev"
```
